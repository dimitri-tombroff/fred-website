Built for Agent Engineers

Fred offers a clean, composable Python SDK to define your own agents — from static document responders to stateful planners and tool users — with full support for async, streaming, and LangGraph workflows.

Dynamic Agent Creation

Fred supports creating agents at runtime — via code or API. Mix declarative YAML with programmatic flows. Your agents can load models, tools, or datasets on the fly, adapting to user or system needs.

Fast to Prototype, Safe to Deploy

Fred is both an innovation lab and a production-ready application. It gives developers a low-friction way to test new agentic ideas, while offering secure deployment (VM, Kubernetes, Helm) for enterprise scenarios.

Designed for Complex Workflows

Fred was built to handle real-world challenges where no single agent is enough. It lets you compose teams of experts — some retrieving documents, others invoking tools — to solve problems together.
